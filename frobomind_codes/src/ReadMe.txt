In order to run the simulation:

$ roscd turftank
$ bin/new_sim

In order to change to different shapes:

change 'field.txt' in turftank/bin/new_sim and in frobomind/fmExecutors/platform/turftank_wpt_nav/src/waypoint_navigation_node.py in line 164


circle.txt --> circle sim
field.txt --> field sim
fild_small.txt --> smaller field sim
rectangle.txt --> rectangle sim 
square.txt --> square sim
